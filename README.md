## Overview

Modern E-commerce Website is a meticulously crafted project that aims to recreate the captivating user experience of [Two Good](https://twogood.com.au/), a ward-winning website. This project showcases the power of HTML, CSS, JavaScript, GSAP for animations, and Locomotive Scroll for smooth scrolling effects, all combined to create a visually stunning and interactive e-commerce platform.
( this website is made in aspect ratio for full screeen so tap f11 for better look of website )

## Features

- **Responsive Design**: The website is designed to be fully responsive, ensuring a seamless experience across all devices.
- **Interactive Animations**: Utilizes GSAP for complex animations, bringing the website to life with smooth transitions and effects.
- **Smooth Scrolling**: Implements Locomotive Scroll for a smooth and customizable scrolling experience, enhancing the overall user interaction.
- **High-Quality Visuals**: The project features high-quality images and graphics, closely mimicking the original website's aesthetic.
- **Performance Optimization**: The website is optimized for performance, ensuring fast load times and a smooth user experience.

## Installation

1. Clone the repository
2. Navigate to the project directory
3. Open the `index.html` file in your preferred web browser to view the website.

## Technologies Used

- **HTML**: For structuring the content of the website.
- **CSS**: For styling the website and ensuring it is responsive.
- **JavaScript**: For adding interactivity and dynamic content.
- **GSAP**: For creating complex animations.
- **Locomotive Scroll**: For implementing smooth scrolling effects.
